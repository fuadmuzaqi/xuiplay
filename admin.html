Generator SQL untuk satu genre dan beberapa lagu

Isi data genre sekali, lalu tambahkan satu atau beberapa lagu. Output akan membuat SQL siap paste ke Turso shell, termasuk `INSERT OR IGNORE` untuk genre dan `INSERT` multi-row untuk lagu. [web:30][web:152]

Input generator

Masukkan genre lalu tambah beberapa lagu dalam genre yang sama. YouTube video ID bisa otomatis diambil dari parameter `v=` pada URL YouTube atau dari format `youtu.be`. [web:155][web:158]

Genre

Data genre akan dibuat satu kali dengan `INSERT OR IGNORE`.

Lagu

Tambah satu atau beberapa lagu. Setiap lagu punya judul, artist, YouTube URL, video ID, dan sort order.

Output SQL

SQL siap paste ke Turso shell SQLite / libSQL

Catatan

  • SQLite mendukung insert banyak row dalam satu statement `VALUES (...), (...), ...`. [web:150][web:159]
  • Turso shell bisa langsung menjalankan SQL dengan `turso db shell nama-db "SQL..."`. [web:30]
  • Kolom `youtube_video_id` tetap penting untuk player berbasis YouTube IFrame API. [web:12]
