Generator SQL untuk genre dan lagu

Isi form, lalu klik generate. Query SQL akan muncul di panel kanan dan bisa langsung kamu copy ke Turso shell.

INSERT genre UPDATE lagu DELETE folder Auto slug & video id

Operasi

Tambah, edit, hapus

Target

genres & tracks

Output

SQL siap paste

Tambah genre

Membuat SQL untuk menambahkan folder genre baru.

Kalau slug dikosongkan, generator akan membuat slug otomatis dari nama genre.

Edit genre

Mengubah nama atau slug genre yang sudah ada.

Hapus genre

Bisa pilih hapus lagunya dulu atau langsung hapus genre saja.

Tambah lagu

Masukkan genre, judul, artist, dan URL YouTube. Video ID akan diambil otomatis dari link bila memungkinkan.

Kolom tambahan seperti cover_url dan accent_color sengaja tidak dipakai supaya schema lebih sederhana.

Edit lagu

Pilih satu acuan utama untuk mencari lagu, lalu isi hanya field yang ingin diubah.

Hapus lagu

Hapus satu lagu berdasarkan ID atau YouTube video ID.

Output SQL

Query siap paste ke Turso shell SQLite / libSQL

Catatan

  • Generator ini hanya membuat SQL, tidak mengeksekusi query.
  • Untuk lagu, video ID bisa otomatis diambil dari URL YouTube.
  • Gunakan mode hapus genre dengan hati-hati bila ingin menghapus semua lagu di dalamnya.
