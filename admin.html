1. Masuk ke Turso shell

Turso CLI menyediakan shell interaktif dan juga bisa mengeksekusi SQL langsung dari command line, jadi ini adalah cara paling cepat untuk mengelola data manual. [web:30][web:31]

Terminal 
turso db shell nama-database-kamu

2. Lihat schema dan isi data

Shell Turso mendukung dot command seperti `.schema` dan `.dump`, sehingga kamu bisa cek struktur database sebelum mengubah data. [web:30][web:35]

Cek schema 
.schema
Lihat data genre 
SELECT id, name, slug, created_at
FROM genres
ORDER BY name ASC;
Lihat data lagu 
SELECT
  id,
  genre_id,
  title,
  artist,
  youtube_video_id,
  sort_order
FROM tracks
ORDER BY genre_id, sort_order, id;

3. Tambah genre

Untuk menambah data baru, gunakan `INSERT INTO`, karena operasi `INSERT` dipakai untuk menambahkan row baru pada SQLite/libSQL. [web:78]

Tambah satu genre 
INSERT INTO genres (name, slug)
VALUES ('Koplo', 'koplo');
Tip: Kalau genre mungkin sudah ada, kamu bisa gunakan `INSERT OR IGNORE` agar query tidak gagal saat nilai unik sudah tersimpan.
Versi aman 
INSERT OR IGNORE INTO genres (name, slug)
VALUES ('Koplo', 'koplo');

4. Tambah lagu ke genre

Untuk memasukkan lagu, kamu tetap memakai `INSERT`, lalu menghubungkan lagu ke genre melalui `genre_id`. Operasi insert dan update data seperti ini adalah pola standar di SQLite/libSQL. [web:78][web:81]

Tambah satu lagu ke genre Koplo

5. Edit genre

Untuk mengubah data yang sudah ada, gunakan `UPDATE` dengan `WHERE` agar hanya row tertentu yang berubah. Struktur `UPDATE` pada SQLite terdiri dari memilih tabel, menetapkan nilai baru, lalu menentukan kriteria baris yang ingin diubah. [web:81]

Ganti nama genre 
UPDATE genres
SET name = 'Koplo Jawa'
WHERE slug = 'koplo';
Ganti slug genre 
UPDATE genres
SET slug = 'koplo-jawa'
WHERE slug = 'koplo';

6. Edit lagu

Kamu bisa mengubah judul, artist, urutan, warna, atau bahkan memindahkan lagu ke genre lain dengan `UPDATE`. Selama memakai `WHERE`, perubahan hanya menyasar record yang cocok. [web:81]

Edit judul dan artist 
UPDATE tracks
SET
  title = 'Kirang (Live Koplo)',
  artist = 'Putri Fedya'
WHERE youtube_video_id = '_VQ0Ei63MlU';
Edit urutan tampil 
UPDATE tracks
SET sort_order = 1
WHERE youtube_video_id = 'FTvPxO1Hfuo';
Pindahkan lagu ke genre lain 
UPDATE tracks
SET genre_id = (SELECT id FROM genres WHERE slug = 'dangdut')
WHERE youtube_video_id = 'jcIsdKJWphs';

7. Hapus lagu

Gunakan `DELETE FROM ... WHERE ...` untuk menghapus record tertentu, karena `DELETE` dipakai menghapus baris yang ada dari tabel. [web:78][web:76]

Hapus berdasarkan ID 
DELETE FROM tracks
WHERE id = 12;
Hapus berdasarkan video ID 
DELETE FROM tracks
WHERE youtube_video_id = 'VmYl-hjwHBw';

8. Hapus genre

Hapus genre hanya setelah memastikan data lagu terkait sudah aman, karena penghapusan data biasanya dilakukan dengan `DELETE` pada baris yang cocok lewat klausa `WHERE`. [web:78]

Hapus semua lagu dalam genre lebih dulu 
DELETE FROM tracks
WHERE genre_id = (SELECT id FROM genres WHERE slug = 'koplo');
Lalu hapus genre 
DELETE FROM genres
WHERE slug = 'koplo';

9. Tips aman sebelum eksekusi

Turso shell juga mendukung dump database ke file SQL dan memuat dump kembali, jadi sebelum edit besar sebaiknya backup dulu. [web:30]

Backup database

Backup 
turso db shell nama-database-kamu .dump > backup.sql

Restore dari dump

Restore 
turso db shell nama-database-kamu < backup.sql
  • Cek dulu row yang mau diubah dengan `SELECT`.
  • Gunakan `WHERE` saat `UPDATE` dan `DELETE`.
  • Backup dulu sebelum bulk edit.
  • Gunakan `youtube_video_id` sebagai acuan cepat saat cari lagu.

Contoh alur tercepat

  1. Masuk ke shell dengan `turso db shell nama-database-kamu`.
  2. Jalankan `SELECT id, name, slug FROM genres;` untuk cek genre.
  3. Tambah genre dengan `INSERT INTO genres ...` bila belum ada.
  4. Tambah lagu dengan `INSERT INTO tracks ...`.
  5. Edit data dengan `UPDATE ... WHERE ...` bila ada salah judul atau urutan.
  6. Hapus data dengan `DELETE ... WHERE ...` bila sudah tidak dipakai.
